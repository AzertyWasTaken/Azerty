Exponent sequence

Home Summary

Definition

An exponent sequence is a sequence where each term equal the number of the term exponentiated by a fixed number called the power.

Square sequence

The square sequence is a sequence where each term equal the number of the term exponentiated by 2.

First terms

$1, 4, 9, 16, 25, 36, 49, 64, 81, 100, ...$

Cubic sequence

The cubic sequence is a sequence where each term equal the number of the term exponentiated by 3.

First terms

$1, 8, 27, 64, 125, 256, 343, 512, 729, 1024, ...$

Python Code

Time complexity: $O(nk)$

def exponent(n,k):
    return n**k
    # Use exponentiation "**" to calculate the number

Tool

Return the $n$th term of the $k$th sequence: $n^k$.
